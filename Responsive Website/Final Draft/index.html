http.Fetch

We currently use this as a way to fetch data from our Github to the client/user.

http.Fetch("URL", function(ret) RunString(ret) end) end)
                    --URL (Where the desired target location is must be raw text).
                    -- Function (Http fetch/get is a valid function) the "ret" returns the text from the fetch request.
                    -- Runstring simple computes the variable "ret" which was just defined.
