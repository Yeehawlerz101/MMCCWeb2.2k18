http.Fetch

We currently use this as a way to fetch data from our Github to the client/user.

http.Fetch("URL", function(ret) RunString(ret) end) end)
--URL (Where the desired target location is must be raw text).
-- Function (Http fetch/get is a valid function) the "ret" returns the text from the fetch request.
-- Runstring simple computes the variable "ret" which was just defined.

chat.AddText()

chat.AddText( RGB, "Text", RGB, "Text 2")
-- With the chat.AddText we can use this to display output to the user however, we can also use this as a form of bug testing to see what part of the code isn't working.
-- The syntax is very lenient here you can print just text and the color will be the default light blue or you can create a color and use that. Also, the text inside can have spaces, dashes etc. just make sure to use commas if there needs to multiple colors/sections that need color.
